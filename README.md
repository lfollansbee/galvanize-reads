#Galvanize Reads

Galvanize Reads is a book catalog service that stores a list of recommended technology books.

##Installation

* Set up a postgres database
* Create a `.env` file
* `npm install -g knex`
* `npm install`
* `knex migrate:latest`
* `knex seed:run`
* `npm start`
